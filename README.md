Contador de Caracteres
Descripción:
Escribe una función o método que tome como entrada una cadena de texto y devuelva un
diccionario/histograma que muestre la frecuencia de cada carácter en el texto, ignorando espacios.

Ejemplo:
Si el texto es "la casa es grande", el resultado debería ser un diccionario 
como {"l": 1, "a": 4, "c": 1, "s": 2, "e": 2, "g": 1, "r": 1, "n": 1, "d": 1}.

Para dar con nuestra solución hicimos lo siguiente:

1- Crear una función que cree una cadena de entrada
2- Cramos un objeto que guarde las frecuencias de cada caracter
3- Recorremos mediante la función cada caracter, ignorando espacios en blanco
4- Contamos la frequencia y almacenamos los rersultados en el objeto
5- Devolvemos los resultados capturados
